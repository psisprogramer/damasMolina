# damasMolina
# Carolina Molina 
El programa implementa un juego de damas con una interfaz visual simple en consola. El diseño del tablero utiliza colores de fondo en las casillas para simular el patrón clásico del juego, mejorando la experiencia del usuario pese a las limitaciones del entorno de texto. Las coordenadas y leyendas facilitan la identificación de posiciones y piezas, permitiendo una interacción más intuitiva. Este enfoque combina funcionalidad y legibilidad, ofreciendo una presentación agradable sin depender de gráficos avanzados.
